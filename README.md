
# Binary tree API

Project to create a binary tree and find near node


## Deployment

To deploy this project: 

clone this repo

with cli:

run: 
```bash
  ./cli
```



## API Reference
Index
#### 
```http
  get /
```

create binary tree
#### 
```http
  get /create
```


find near node
#### 
```http
  Post /binarytree
```
Json example:
    {
        "input": "['70,84,85','70,84,78,80']"
    }
